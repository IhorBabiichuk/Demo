Deploying Azure Function

1. Prepare your environment

  • Install Azure CLI
  • Install Azure Functions Core Tools
  • Install .NET 8 SDK (if your function uses .NET)

2. Log in to Azure

az login

3. Create Function App

az functionapp create \
  --resource-group MyGroup \
  --consumption-plan-location westeurope \
  --runtime dotnet \
  --functions-version 4 \
  --name my-function-app \
  --storage-account mystorage123

4. Deploy to Azure

func azure functionapp publish my-function-app

Once deployed, copy the Function URL and use it as a webhook or API endpoint inside Power Automate or Power Apps.

Installing Power Pages Solution

1. Import solution

  • Go to Power Apps → Solutions
  • Select Import
  • Choose the managed solution ZIP
  • Wait until import completes

2. Configure Environment Variables

  • Open Solutions → Environment Variables
  • Set values for required API URLs, keys, or configuration settings

3. Publish changes

Open Power Pages site → Sync configuration → Publish.

4. Optional: Enable integration

  • Activate Power Automate flows
  • Enable site permissions
  • Validate table permissions
