## üß≠ Using Call Report AI

### Creating a New Call Report
1. Open the **Call Report AI** application.
2. Select **Create Call Report**.
3. Fill in the call details:
   - Caller / Contact  
   - Subject  
   - Call Duration  
   - Outcome  
   - Notes  

### Generating AI Insights
If enabled, AI can analyze notes and generate:
- summary  
- recommended follow-up actions  
- sentiment indicators
