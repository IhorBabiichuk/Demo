Installing Power Apps Solution

1. Import solution

  • Go to Power Apps → Solutions
  • Select Import
  • Choose the managed solution ZIP

2. Configure Environment Variables

  • Open Solutions → Environment Variables
  • Set values for required API URLs, keys, or configuration settings
  • Wait until import completes

3. Publish changes

Open Power Pages site → Sync configuration → Publish.

4. Optional: Enable integration

  • Activate Power Automate flows
