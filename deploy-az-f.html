Deploying Azure Function

1. Prepare your environment

  • Install Azure CLI
  • Install Azure Functions Core Tools
  • Install .NET 8 SDK (if your function uses .NET)

2. Log in to Azure

az login

3. Create Function App

az functionapp create \
  --resource-group MyGroup \
  --consumption-plan-location westeurope \
  --runtime dotnet \
  --functions-version 4 \
  --name my-function-app \
  --storage-account mystorage123

4. Deploy to Azure

func azure functionapp publish my-function-app

Once deployed, copy the Function URL and use it as a webhook or API endpoint inside Power Automate or Power Apps.
